# Http Https

# http

影响http请求的因素：带宽、延迟

延迟的触发原因：

  1. 浏览器阻塞，当同域名下的连接超过浏览器的限制，就会阻塞连接
  2. DNS查询，有DNS缓存可以减少该部分时间
  3. 建立连接，要到第三次握手才真正建立起连接，高延迟环境下会使连接过程更慢

# https

http和https的区别：

  1. 可防止运营商劫持
  2. 需要证书
  3. 运行在TTL/TLS上，TTL/TLS运行在TCP上，内容都是经过加密的
  4. http端口为80，https端口为443
